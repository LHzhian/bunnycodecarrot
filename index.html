Game Introduction

BunnyCodeCarrot is a programming game designed specifically for children. Through the fun way of helping the cute bunny pull carrots, children learn programming logic and develop computational thinking skills while playing.

Suitable age: 6-12 years old

Why 6-12 year old children need to learn computational thinking

In the digital age, computational thinking has become one of the essential core abilities for children. The age of 6-12 is a critical period for children's logical thinking development. By learning programming concepts, children can develop problem decomposition, pattern recognition, algorithm design, and abstract thinking abilities. BunnyCodeCarrot transforms abstract programming concepts into a lively and interesting bunny carrot-pulling game through gamified teaching methods, allowing children to understand programming logic such as loops and conditional judgments in practice. Our game design meets children's cognitive characteristics, and through the drag-and-drop command card method, it lowers the threshold for programming learning, allowing every child to easily get started and fall in love with the charm of logical thinking.

How to Play

  1. Select the required commands from the command cards on the right
  2. Drag the commands to the command queue in the middle
  3. Click the "Start" button to watch the bunny execute the commands
  4. Help the bunny successfully pull the carrot by adjusting the order and combination of commands
  5. Use the "Loop" command to repeat a series of operations

Educational Value

  • Cultivate logical thinking ability
  • Learn basic programming concepts
  • Improve problem-solving ability
  • Cultivate patience and perseverance
  • Stimulate interest in programming
